# Projeto Aplicado II - Dengue

Projeto da disciplina Projeto Aplicado II com foco em análise textual e classificação de boletins epidemiológicos sobre dengue, utilizando dados públicos do Ministério da Saúde.

## Objetivo
Desenvolver uma base textual estruturada e um protótipo analítico capaz de organizar e classificar documentos epidemiológicos sobre dengue para apoio ao monitoramento em saúde pública.

## Fonte dos dados
- Ministério da Saúde
- Documentos públicos oficiais sobre dengue
- Boletins epidemiológicos, informes e comunicados técnicos

## Estrutura do repositório
- `data/`: bases de dados brutas e tratadas
- `docs/`: documentos da disciplina e materiais de apoio
- `notebooks/`: análises exploratórias e experimentos
- `src/`: scripts do projeto
- `reports/`: relatórios e entregas
